If the machine engrave wavy line.
Solution: 
1. Please check whether the timing belt is loose.
2. There should be no gap between the wheels and aluminum profile.
3. Please check whether the wheels is too loose or too tight to move, the wheels should move smoothly.
4. Please check whether all the wheels not in same level. 
5. Please check whether there is wheel broken, if broken or bad quality, it need to replace.
